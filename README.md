# Azure Custom Deployment (AZ-104 Lab Project)

## Overview
This project documents my custom Azure deployment lab completed through ACloudGuru.  
It demonstrates creating and deploying a virtual machine and associated resources in Azure.

- Created a resource group in Azure
- Deployed a VM using the Azure Portal
- Configured network settings and storage
- Verified deployment in Azure Portal
